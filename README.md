## Installation

If you have the node package manager, npm, installed:

```shell
npm install sfinktah/QuickScrape
cd node_modules/quickscrape
npm test
```

Or via git

```shell
git clone https://github.com/sfinktah/QuickScrape.git
cd QuickScrape
npm install
npm test
```
